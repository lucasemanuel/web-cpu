# web-cpu

## Trabalho de Arquitetura e Organização de Computadores

O Web-cpu executa instruções básicas de uma cpu.

Autor: [Lucas Emanuel](https://github.com/lucasemanuel8)

*******

## Documentação
Instruções programadas:
* AND
* NAND
* OR
* XOR
* NOR
* XNOR
* ADD
* COMP [Complemento de 2]
